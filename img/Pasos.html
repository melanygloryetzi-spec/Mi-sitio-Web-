🌸 Pasos para crear el sitio web personal 🌸

Este sitio fue realizado como parte de una tarea del curso Introducción a los Sistemas de Cómputo. A continuación, se describen los pasos realizados para crear la página web personal.

  • 1.- Creé una carpeta con mi número de carnet como nombre: Sitio Web 1590-26-6896
  • 2.- Utilicé Bloc de notas para crear los archivos HTML del sitio.
  • 3.- Diseñé la estructura principal usando la etiqueta <frameset> para dividir el menú y el contenido.
  • 4.- En el archivo menu.html coloqué enlaces hacia las diferentes secciones del sitio.
  • 5.- Agregué una hoja de vida en hoja_vida.html.
  • 6.- Creé una tabla de notas de los primeros parciales con las materias cursadas en notas.html.
  • 7.- Coloqué imágenes dentro de una carpeta llamada imagenes/ y las mostré en galeria.html.
  • 8.- Inserté un video de YouTube usando la etiqueta <iframe> en video.html.
  • 9.- Agregué un juego simple hecho con JavaScript en juego.html.
  • 10.- Subí todos los archivos a GitHub y habilité GitHub Pages para que el sitio estuviera disponible en línea.
  • 11.- Utilicé mi número de carnet como parte de la URL para identificar mi sitio web.
